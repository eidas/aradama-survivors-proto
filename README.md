# aradama-survivors-proto

アニメ「刀使ノ巫女」ファンメイドの 2D サバイバーライク(ヴァンサバライク)プロトタイプ。
湧き続ける荒魂(あらだま)を、刀使(とじ)が御刀で斬り倒し続ける。

## 設計ドキュメント

作成順に読むことを想定しています。

1. [ゲームデザイン概要](docs/01_game_design_overview.md) — コンセプト、コアループ、キャラ・能力・敵の全体像、スコープ
2. [技術概要設計](docs/02_technical_overview.md) — 技術選定(Phaser 3 + TypeScript + Vite)、アーキテクチャ方針、性能目標
3. [ゲームデザイン詳細](docs/03_game_design_detail.md) — 能力・キャラ・敵・ウェーブ・強化プールの数値仕様
4. [技術詳細設計](docs/04_technical_detail.md) — ディレクトリ構成、データスキーマ、システム実装、テスト計画、マイルストーン

## 開発

```bash
npm install
npm run dev      # 開発サーバ(デバッグキー有効: O=300体湧き K=+60秒 I=無敵)
npm test         # ロジック単体テスト(Vitest)
npm run build    # 型チェック + プロダクションビルド(dist/)
npm run preview  # ビルド結果の確認
```

技術スタック: Phaser 3 + TypeScript + Vite(選定理由は [技術概要設計](docs/02_technical_overview.md))
